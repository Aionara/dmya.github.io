# 🎓 课堂点名工具（带语音播报）

专为教师课堂互动设计的点名工具，支持语音播报功能，有效保护嗓音。无需安装、零依赖、无需联网，浏览器打开即用（建议按 F11 进入全屏模式）。

---

## ✨ 功能亮点

- ✅ 零依赖，无需安装
- ✅ 离线运行，无需网络
- ✅ 浏览器即开即用
- ✅ 系统语音播报（支持 Chrome 和 Firefox）
- ✅ 简洁界面，流畅操作

---

## 🧩 五大核心功能

| 功能名称       | 描述                                                                 |
|----------------|----------------------------------------------------------------------|
| 上课提问       | 随机抽取学生，支持权重记忆，提升互动效率                             |
| 顺序点名       | 按名单顺序循环点名，确保每位学生都被覆盖                             |
| 随机点名       | 完全随机抽取，公平无偏见，适合日常点名                               |
| 快速连抽       | 一次抽取多名学生，适合小组活动或分组讨论                             |
| 计时器模式     | 倒计时功能（可自定义时间），营造紧张氛围，提升课堂专注度             |

---

## 🆕 新增功能

- 🖥️ 窗口自适应：自动适配不同屏幕尺寸
- 🗣️ 自定义叫号文本：可设置个性化播报内容

---

## 🔊 语音播报说明

- 默认启用系统语音播报功能
- Chrome 浏览器首次使用需授权语音权限
- 可通过右上角语音按钮关闭播报功能

---

## 🚀 使用方式

1. 打开支持的浏览器（推荐 Chrome 或 Firefox）
2. 双击打开 `index.html` 文件
3. 按 F11 进入全屏模式（可选）
4. 导入学生名单，开始点名互动！
