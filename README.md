# 📋 点名网站

一个简洁高效的在线点名系统，适用于教师、培训师或活动组织者快速记录和管理出勤情况。

---

## 🚀 项目简介

本项目旨在提供一个轻量级、易用的点名工具，支持以下功能：

- 课堂花样点名（固定单点，随机点，一点多个，课堂点到）快速记录出勤状态（出勤、迟到、缺席）
- 生成统计报表（CSV/Excel）
- 新增功能：1、窗口自适应 2、自定义叫号文本 3、功能面板折叠功能
系统界面简洁，操作流畅，适合日常教学或活动管理使用。
